# 🔴 LIGUE-4 (Connect Four) 🟡

Um clássico jogo de estratégia em duas dimensões desenvolvido em Python, com animações suaves e interface de terminal interativa.

## 📋 Sobre o Jogo

Connect Four (também conhecido como Ligue-4) é um jogo de tabuleiro para dois jogadores onde o objetivo é ser o primeiro a formar uma sequência de quatro peças consecutivas na horizontal, vertical ou diagonal.

## 🎮 Características

- ✨ **Animações visuais** - As peças "caem" suavemente até a posição escolhida
- 🎯 **Verificação automática** - Detecção de vitória em todas as direções
- 🚦 **Validação de jogadas** - Impede movimentos inválidos ou colunas cheias
- 🏆 **Tela de vitória** - Mensagem personalizada para o jogador vencedor
- 🤝 **Detecção de empate** - Identifica quando o tabuleiro está completamente preenchido

## 🕹️ Como Jogar

1. Execute o jogo no terminal
2. O Jogador 1 joga com o símbolo **●**
3. O Jogador 2 joga com o símbolo **○**
4. Em cada rodada, digite um número de **1 a 7** para escolher a coluna
5. O jogo continua até alguém formar 4 peças em linha ou o tabuleiro encher

### Regras Básicas
- As peças caem sempre para a posição mais baixa disponível na coluna
- Você não pode escolher uma coluna que já está cheia
- Vitória é declarada quando um jogador consegue 4 peças consecutivas:
  - Horizontal (↔️)
  - Vertical (↕️)
  - Diagonal (↗️↘️)
